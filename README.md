# Recipe-Management-System
Applied Software Engineering (SIT725) Group Project

<img width="100" height="100" alt="rms logo" src="https://github.com/user-attachments/assets/efe121b9-b1d1-4a7e-a5c1-95e0f864f8e1" />


A simple and efficient application for creating, organizing, and managing recipes. It allows users to store ingredients, instructions, categories, and images, making it easy to build a personal cookbook or manage recipes for a restaurant, blog, or home use.

Features

Create, Read, Update, Delete (CRUD) Recipes

Categorize Recipes (e.g., Breakfast, Desserts, Vegetarian)

Ingredient Management

Step-by-step Instructions

Search & Filter Recipes

Upload Photos

User Authentication (optional depending on implementation)

Responsive UI
